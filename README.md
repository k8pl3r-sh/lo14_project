
# Installation

Install `jq` with `apt install jq`


# Use
`./rvsh` (you can eventually add it as a symlink)

Options:
`-connect`:
	`hostname user`


`-admin`: (login)
	`host`
		`-a <hostname>`
		`-r <hostname>`
	`user <user>`
		`-a <hostname1 hostname2 ...>`
		`-r <hostname ...>`
	`wall`
		`msg`
		`-n msg`
