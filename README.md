
# Installation

Install `jq` with `apt install jq`


# Use
`./rvsh` (you can eventually add it as a symlink)

Options:
`-connect`:
	`hostname user`


`-admin`: (login)
`host`
	`-a <hostname>` or `-r <hostname>`
`user <user>` (nologin by default)
`-a <hostname1 hostname2 ...>`
`-r <hostname ...>`
`wall` `msg` or `-n msg`
